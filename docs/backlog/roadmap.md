# ロードマップ

## ビジョン

**最終目標**: 法人向けCASB（Cloud Access Security Broker）
- AIサービスの利用状況を可視化し、プライバシーリスクを管理
- 端末完結型（サーバーレス）のブラウザ拡張機能
- プライバシー保護（データは端末に留まる）
- 「検出→可視化→ポリシー生成」のフロー

## Phase 1: MVP強化 ✅ 完了

**目標**: 個人ユーザー向け価値検証

### 実装済み
- サービス検出（ログインページ、認証）
- CSP監査
- AIプロンプト監視（基本）
- PII検出（プロンプト内の個人情報検出）
- リスクスコアリング（AIプロンプトの機密性レベル自動評価）
- エクスポート機能（監査ログのCSV/JSON出力）
