# プライバシーポリシー

最終更新: 2025年1月

## 基本方針

Pleno Auditは**プライバシーファースト**の設計思想に基づいています。

- **デフォルト**: すべての処理は端末内で完結し、外部サーバーへのデータ送信は行いません
- **オプトイン**: 一部の機能は明示的なユーザー同意を得た場合のみ外部通信を行います

## データ収集

### 収集するデータ

Pleno Auditは以下のデータを**端末内でのみ**収集・保存します：

| データ種別 | 目的 | 保存場所 |
|-----------|------|---------|
| CSP違反ログ | セキュリティ監視 | IndexedDB (ローカル) |
| ネットワークリクエスト | 脅威検出 | IndexedDB (ローカル) |
| AIプロンプト | Shadow AI検出 | IndexedDB (ローカル) |
| サービス検出情報 | CASB機能 | chrome.storage.local |
| イベントログ | 監査証跡 | IndexedDB (ローカル) |

### 外部送信しないデータ

- 閲覧履歴
- パスワード
- クッキー
- フォーム入力内容
- AIプロンプトの内容

## オプトイン機能

以下の機能はユーザーの明示的な同意を得た場合のみ有効化されます：

### 1. サーバー連携（デフォルト: OFF）

組織での一元管理を目的とした機能です。

- **送信データ**: 匿名化されたセキュリティイベント、ポリシー設定
- **送信先**: ユーザーが指定したエンドポイントのみ
- **制御**: 設定画面からいつでもON/OFF可能

### 2. RDAP問い合わせ（デフォルト: OFF）

NRD（新規登録ドメイン）検出の精度向上を目的とした機能です。

- **送信データ**: ドメイン名のみ
- **送信先**: 公開RDAPサーバー
- **制御**: 設定画面からいつでもON/OFF可能

### 3. CSP違反レポート（デフォルト: OFF）

CSP違反の外部レポートを目的とした機能です。

- **送信データ**: CSP違反レポート（report-uri形式）
- **送信先**: ユーザーが指定したエンドポイントのみ
- **制御**: 設定画面からいつでもON/OFF可能

### 4. 外部連携（Slack/Webhook）（デフォルト: OFF）

セキュリティイベントの通知を目的とした機能です。

- **送信データ**: アラート内容（ドメイン名、イベント種別）
- **送信先**: ユーザーが設定したWebhook URL
- **制御**: 設定画面からいつでもON/OFF可能

## データ保持

- **デフォルト保持期間**: 180日
- **設定可能範囲**: 無期限〜365日
- **削除**: 設定画面から全データを即座に削除可能

## 第三者への提供

Pleno Auditは収集したデータを第三者に販売・提供しません。
